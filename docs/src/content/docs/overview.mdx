---
title: Übersicht über die Rollen
description: Die verschiedenen Rollen auf einem Blick.
---
### Nicht angemeldete Nutzer
Nutzer, die nicht angemeldet sind, können nach Tierprofilen und NGO-Profilen filtern und suchen, haben aber keinen Zugriff auf die Profilansicht eines einzelnen NGO-Mitglieds.  
Sie haben aber Zugriff auf die Detailansicht von Tierprofilen und NGO-Profilen.

### Tierhalter
Nutzer, die als Tierhalter angemeldet sind, können am Vermittlungsprozess für ein Tier teilnehmen. Tierhalter können auf einem Tierprofil einen Button klicken, mit denen Sie eine E-Mail an das verantwortliche NGO-Mitglied schreiben können.  
Die Tierhalter werden dann von dem verantwortlichen NGO-Mitglied kontaktiert, wenn Sie als Kandidat in Frage kommen.
Darüber hinaus können Tierhalter Tiere auch zu ihrer Merkliste hinzufügen. 

### NGO-Administrator
Bei der Erstellung eines NGO-Profils schickt der NGO-Administrator ein Verifikationsdokument für die NGO mit, welches vom Webseiten-Admin genehmigt werden muss.  
NGO-Administratoren haben die Rechte für die Bearbeitung, sowie Löschung des NGO-Profils.

### NGO-Mitglied
NGO-Mitglieder können Tierprofile erstellen und veröffentlichen. 
Sie können andere NGO-Mitglieder zu ihrer NGO einladen, damit Sie sich leichter registrieren können.  
Alle Tierprofile können in einer Ansicht "My Animals" angezeigt werden.

### Webseiten-Administrator
Ein Webseiten-Admin genehmigt oder lehnt das Verifikationsdokument einer NGO ab.
